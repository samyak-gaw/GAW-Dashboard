☁️ JSONBin Cloud Sync Setup
Connect to JSONBin.io (free) so every upload on desktop instantly syncs to all phones. Follow steps exactly — 2 minutes.

1️⃣ Go to jsonbin.io → Sign up free (no credit card needed)

2️⃣ Sidebar → Bins → Create a Bin → name it GAW-Apr2026 → Save → copy the Bin ID (looks like 6818abc123...)

3️⃣ Sidebar → API Keys → expand X-ACCESS-KEY → copy that key (starts with $2a$10$...)

⚠️ Do NOT use X-Master-Key — causes 401 errors

4️⃣ Paste both below and click Save & Connect

5️⃣ Share this dashboard file to your team — they paste the sync code below

🐙 GitHub Auto-Deploy Sync (Recommended for reliability)
Pushes data as a JSON file to your GitHub repo → triggers a Netlify redeploy automatically. Every phone always pulls from Netlify — no API keys needed for viewers, completely free and unlimited.

One-time setup (5 mins):

  1. Go to github.com — create a free account if you don't have one
  2. Create a new repo (e.g. gaw-dashboard) and upload your index.html there
  3. Connect repo to app.netlify.com → Add site → Import from GitHub → select your repo
  4. Create a GitHub Personal Access Token → github.com/settings/tokens → Tokens (classic) → Generate → check repo scope → copy it
  5. Enter your token, GitHub username, and repo name below → click Save GitHub Config
  6. Done. Every upload auto-pushes data to GitHub → Netlify rebuilds → phones get fresh data within ~30 seconds
