☁️ Cloud Sync Setup

Connect to JSONBin.io (free) so every upload on desktop instantly syncs to all phones. Follow steps exactly — 2 minutes.

  • Go to jsonbin.io → Sign up free (no credit card needed)
  • In sidebar click Bins → Create a Bin → name it GAW-April2026 → Save → copy the Bin ID shown (looks like 6818abc123…)
  • In sidebar click API Keys → expand X-ACCESS-KEY section → copy that key (starts with $2a$10$…). ⚠ Do NOT use X-Master-Key — causes 401 errors
  • Paste both below and click Save & Connect
  • Share this dashboard file to your phone/team — they paste the same two values

⚡ Option B — GitHub + Netlify Sync (Recommended for reliability)

🐙 GitHub Auto-Deploy Sync

This method pushes data as a JSON file directly to your GitHub repo, which triggers a Netlify redeploy automatically. Every phone/team member always pulls from Netlify — no API keys, no 100KB limits, completely free and unlimited.

One-time setup (5 mins):
  1. Go to github.com → create a free account if you don't have one
  2. Create a new repo (e.g. gaw-dashboard) and upload your index.html there
  3. Connect repo to Netlify: app.netlify.com → Add site → Import from GitHub → select your repo
  4. Create a GitHub Personal Access Token: github.com/settings/tokens → Tokens (classic) → Generate → check repo scope → copy it
  5. Enter your token, GitHub username, and repo name below → click Save GitHub Config
  6. Done. Every upload auto-pushes data to GitHub → Netlify rebuilds → phones get fresh data within ~30 seconds
