🐙 GitHub Auto-Deploy Sync
This method pushes data to a GitHub repo (+ Netlify redeploys automatically every 60 seconds). Admin only. Every phonemate always calls freshly — no API keys, completely free and unlimited.

One-time setup (5 mins):

  1. Go to github.com — create a new repo (e.g. gaw-dashboard), and upload your index.html if not uploaded your folder/file.
  2. Create a new repo (e.g. gaw-dashboard) → choose "Private" if you want → click Create
  3. In that repo create a .github/workflows/deploy.yml file → get template here
  4. Create a GitHub Personal Access Token → Tokens (classic) → Generate → check repo scope → copy it
  5. Enter your token, GitHub username, and repo name below → click Save GitHub Config
  6. Done. Every upload auto-pushes data to GitHub → Netlify redeploys → all phonemates = fresh data always.
